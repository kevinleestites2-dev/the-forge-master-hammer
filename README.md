## The Strike Pipeline

```
Intent → ZEUS (judge) → THOR (bridge) → CLAW (strike) → Result
```

| Layer | Name | Function |
|-------|------|----------|
| **Will** | ZEUS | Judges whether an intent is worthy |
| **Bridge** | THOR | Routes intent into an execution plan via Bifrost |
| **Strike** | CLAW | Executes the plan and hits the target |

## Features

- **Zeus/Thor/Claw Pipeline** — judge, bridge, strike. Never misses.
- **Storm Mode** — bypasses Zeus, full autonomous override
- **8 Brain Modes** — auto-selects based on task context
- **Liquid Memory** — persistent cross-session recall
- **StateBus** — real-time nervous system across components
- **Task Queue** — priority-sorted, auto-retrying
- **Self-Comitting** — pushes its own progress to GitHub
- **Dark Forest Mode** — `DARK_FOREST=true` — silent operation
- **Checkpoint System** — Apollo-X style, survives restarts
- **SkillClaw** — self-expanding skill library
- **Dream Cycle** — agent ponders what to do next autonomously

## Quick Start

```bash
# Set your keys
export DEEPSEEK_API_KEY=your_key
export GROQ_API_KEY=your_key
export GEMINI_API_KEY=your_key
export GITHUB_TOKEN=your_token
export GITHUB_REPO=kevinleestites2-dev/the-forge-master-hammer

# Install dependencies
pip install aiohttp

# Run — the Forgemaster heartbeat loop
python forgemaster.py
```

## CLI Commands (standalone mode)

```bash
# Forge a mission on the anvil
python forgemaster.py forge "Deploy the swarm on Base chain" 1

# Strike — execute an action
python forgemaster.py strike "Research latest DeFi yields"

# Check the forge
python forgemaster.py status

# Dream of what's next
python forgemaster.py dream

# Full forge report
python forgemaster.py report
```
